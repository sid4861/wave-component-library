Avatar

The avatar represents a user, and displays the profile picture, initials or fallback icon. It comes in three sizes, small, medium, large

Usage and examples

Just use the img tag with the appropriate class

Small Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-small">

Medium Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-medium">

Large Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-large">

Alert

Alerts provide feedback messages for user actions

Usage and examples

Use the div tag with the approprate alert class. There are 5 types of alert classes available - primary, success, danger, warning, and info

Primary Alert

                
                    < div class="alert alert-primary">
                    <span class="alert-close" onclick="this.parentElement.style.display='none';">×</span> 
                    A simple primary alert—check it out!
                    </div >
× A simple primary alert—check it out!

Success Alert

                
                    < div class="alert alert-success">
                    <span class="alert-close" onclick="this.parentElement.style.display='none';">×</span>
